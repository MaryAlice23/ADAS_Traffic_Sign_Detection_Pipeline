 
# 🚗 ADAS Traffic Sign Detection & Classification

## 📌 Project Overview
This project implements a complete **ADAS (Advanced Driver Assistance System) pipeline** for traffic sign detection and classification.

It consists of:
1. Sign Detection (OpenCV + YOLO)
2. Sign Classification (CNN / MobileNetV2 / ResNet50)

---

## 🔍 Features
- Detect traffic signs from full road images
- Classify into 43 categories (GTSRB dataset)
- Real-time prediction using Streamlit
- Top-3 prediction display
- Robust detection using multi-stage pipeline

---

## 🧠 Models Used
- Custom CNN
- MobileNetV2 (Transfer Learning)
- ResNet50 (Best Accuracy)

---

## ⚙️ Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- Streamlit
- Ultralytics YOLO
